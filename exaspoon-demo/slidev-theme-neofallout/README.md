## Usage

Install with Claude CLI:

```
claude write-file slidev-theme-neofallout/package.json "<contents>"
claude write-file slidev-theme-neofallout/index.js "<contents>"
claude write-file slidev-theme-neofallout/styles.css "<contents>"
claude write-file slidev-theme-neofallout/layouts/default.vue "<contents>"
claude write-file slidev-theme-neofallout/README.md "<contents>"
```

Then install:

```
npm install ./slidev-theme-neofallout
```

Use in your `slides.md`:

```yaml
---
theme: slidev-theme-neofallout
---
```

Enjoy your wasteland-themed presentation.
